Tentang Saya

Saya adalah seorang pengembang web dari Indonesia dengan passion untuk menciptakan solusi digital yang inovatif, efisien, dan estetik. Saya menikmati proses membangun aplikasi web dari awal hingga akhir.

Keahlian saya mencakup pengembangan Frontend dan Backend. Saya terampil dalam menggunakan Node.js untuk membangun API dan layanan backend yang tangguh, serta HTML, CSS (Tailwind CSS), dan JavaScript untuk menciptakan antarmuka pengguna yang interaktif dan responsif.

Di luar pekerjaan, saya gemar mengeksplorasi teknologi baru dalam ekosistem Node.js, terus belajar, dan berkontribusi pada proyek open-source. Saya percaya pada kekuatan kode untuk memecahkan masalah dan menciptakan pengalaman pengguna yang luar biasa.
