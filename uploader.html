Upload File via HTML Form

Catatan Penting: Menggunakan form langsung seperti ini akan mengirim file ke Imgur, tetapi Anda tidak akan bisa membaca respons (seperti URL gambar yang diunggah) di halaman ini karena kebijakan keamanan browser (Same-Origin Policy), kecuali jika Imgur mengizinkannya melalui CORS.

Respons dari Imgur (jika ada) kemungkinan akan ditampilkan di tab baru jika Anda menggunakan `target="_blank"`.
